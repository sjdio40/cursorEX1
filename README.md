# 📷 이미지 뷰어 & 슬라이드쇼 프로그램

이 프로그램은 지정된 폴더의 이미지를 **버튼 클릭**으로 탐색하고, 선택한 이미지만 슬라이드쇼로 재생할 수 있습니다.

---

## 🎯 기능
✅ **이미지 탐색**
   - `⬅ 이전` 버튼: 이전 이미지 보기
   - `➡ 다음` 버튼: 다음 이미지 보기

✅ **이미지 선택**
   - `✅ 선택/해제` 버튼: 현재 이미지를 체크하거나 해제

✅ **슬라이드쇼 기능**
   - `🎬 슬라이드 시작` 버튼: 체크한 이미지만 3초 간격으로 슬라이드쇼 실행
   - `🛑 슬라이드 중지` 버튼: 슬라이드쇼 정지

---
